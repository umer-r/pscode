## PScode
***PScode Version 1.0***

# Description
PowerShell script to PScode encodes and decodes regular strings to Base64, Hexa, Binary and vice versa.

# Usage
### On Windows:
```
pscode.ps1 ([-Help] | [-Method] Encode|Decode [-Algo] Base64|HEX|Binary [-Value] Reg|Encoded String)
```
